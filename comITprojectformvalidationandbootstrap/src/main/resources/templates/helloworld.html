Welcome to the KidnApp!

The purpose of this application is to allow quick and accurate comparisons between drugs and a patient's measured or estimated renal function (as eGFR or CrCl). Please keep in mind this tool is meant to be used to better guide decision-making for healthcare providers, and not as a definitive recommendation for a given patient. Please refer to the associated references for more context. To use, enter the patient's renal function and select a drug in question from the list below. Note that the list is not exhaustive.

Please enter a value between 0-140mL/min.
